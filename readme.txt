How to configure development environment:
1. With Visual Studio
	a. Install Visual Studio (Do not install EXPRESS edition)
	   For QT development you must install the C++ feature
	b. Install QT at least 5.2.1 version (be carefull that downloaded QT disributive can be used with installed visual studio). 
	c. Install QT VS add-in 1.2.4 or later
